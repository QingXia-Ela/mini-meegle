# Mini meegle

功能列表：
- 用户登录/注册
- 加入/创建公司
- 工作台
  - 目前有多少未完成需求
  - 已完成多少需求
  - 创建了多少需求
  - 正在关注的需求

空间：
- 导航顺序

工作项：
- 条目列表
  - 全部
  - 最近浏览
  - 我创建的
  - 我参与的
  - 我关注的
- 条目元数据
  - 字段可配置（crud）
  - 字段种类
    - 名称，描述，采用流程，相关人员列表（固定）
- 条目详细情况
  - 节点流程
    - 子任务
  - 基本信息（与元数据一致，页面布局由配置决定）
  - 关联工作项条目（任何关联均可）
  - 节点详情
  - 评论

空间配置：
- 空间信息
  - 名字
  - 导航顺序
  - 图标
  - 删除空间
  - 工作项管理
- 工作项
  - 基本信息
    - 增：
      - 流程模式（创建后不可修改）
        - 节点模式
        - 状态模式
      - 名字，描述
      - 图标与颜色
  - 字段管理（与元数据相关）
  - 页面布局（管理新建页、条目详情页布局）
  - 流程管理
  - 需求模板管理

权限系统：
- 组织所有者
- 管理员
- 普通员工

通知：
- 留言消息
- @消息
- 系统消息
