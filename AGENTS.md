# 项目概述

这是一个基于 React + Nestjs 的项目管理平台，使用 MYSQL，sequelize-typescript 做数据库管理。前端使用 tailwind, zustand, react-router(Data Mode)

## 关键目录

以本文件为当前目录

- `./src` 前端代码
- `./src-backend` 后端代码

## 启动项目

- 前端：`npm run dev`
- 后端：`cd src-backend && npm run start:dev`
